# Helpers

> Apply helper classes to almost any element, in order to alter their style

```tsx
const Block: React.FC<unknown> = ({ children, ...props }) => (
	<div {...props}>{children}</div>
);
const WithHelper = withHelpersModifiers(Block);
```
