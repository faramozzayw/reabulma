## Other <small>[Bulma reference](https://bulma.io/documentation/helpers/other-helpers/)</small>

Other useful Bulma helpers

### ClearFix

> Fixes an element's floating children

#### Property: `isClearFix`

#### Example:

```tsx
<WithHelper isClearFix />
```

### Pulled

> Moves an element to the **left** or **right**

#### Property: `isPulled`

#### Example:

```tsx
<WithHelper isPulled="left" />
<WithHelper isPulled="right" />
```

### Overlay

> Completely **covers** the first positioned parent

#### Property: `isOverlay`

#### Example:

```tsx
<WithHelper isOverlay />
```

### Clipped

> Adds overflow **hidden**

#### Property: `isClipped`

#### Example:

```tsx
<WithHelper isClipped />
```

### Radiusless

> Removes any **radius**

#### Property: `isRadiusless`

#### Example:

```tsx
<WithHelper isRadiusless />
```

### Shadowless

> Removes any **shadow**

#### Property: `isShadowless`

#### Example:

```tsx
<WithHelper isShadowless />
```

### Unselectable

> Prevents the text from being **selectable**

#### Property: `isUnselectable`

#### Example:

```tsx
<WithHelper isUnselectable />
```

### Clickable

> Applies `cursor: pointer !important` to the element.

#### Property: `isClickable`

#### Example:

```tsx
<WithHelper isClickable />
```

### Relative

> Applies `position: relative` to the element.

#### Property: `isRelative`

#### Example::

```tsx
<WithHelper isRelative />
```
